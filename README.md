This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started

First, install dependencies:

```bash
yarn install
```

Then run for development

```bash
yarn dev
```

Run tests

```bash
yarn cypress
```

Run linter

```bash
yarn lint
```

### Challenge

Rick and Morty app.

##### Requirements:

- React (NextJS)
- Express (used Firebase)
- No style libraries
- Fav list
- Store in a repo
- Quality first
- Store in DB (Firebase)

##### Functionalities:

- Login
- List characters
- Detail characters
- 404 page
- Fav list

##### Extra points:

- tests
- register
- backend pagination
- preload images and loader
